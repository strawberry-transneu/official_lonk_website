# graphics and art

- some graphics created through using stock photos and public domain images 
- OR hand-drawn with the help of free graphic editing / drawing software [firealpaca](https://firealpaca.com/).
- main font is [cmu typewriter](https://www.cufonfonts.com/font/cmu-typewriter-text): cyrillic & commercially friendly !
- many of fonts were found using [1001 fonts](https://www.1001fonts.com/), as well as [ukrfonts](https://ukrfonts.com/index.php?v=19): specialized library for ukrainian-friendly fonts!
- made with love
